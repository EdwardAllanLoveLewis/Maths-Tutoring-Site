Algebraic Techniques

Index Laws

An index of a number represents repeated multiplication. Other words for index include, power or exponent.

For example:

  • $$4^3=4\times4\times4=64$$
  • $$a^5=a\times a\times a\times a\times a$$
Note, for the above example a is the base and 5 is the index

There are a few general laws, known as the index laws which make simplifying calculations with indices quite simple. These laws are listed below.

Index laws

  • $$a^m \times a^n = a^{m+n}$$
  • $$a^m \div a^n = a^{m-n}$$
  • $$(a^m)^n=a^{mn}$$
  • $$(ab)^n=a^nb^n$$
  • $$(\frac{a}{b})^n=\frac{a^n}{b^n}$$
Note, these laws work for any m and n including fractions and negative numbers

Zero and Negative Indices

  • $$x^0=1$$
  • $$x^{-n}=\frac{1}{x^n}$$

    • For any base with an index of 0, the result is equal to 1. For negative indices the result is the reciprocal raised to the positive power.

    Fractional Indices

    • $$a^{\frac{1}{n}}=\sqrt[n]{a}$$
    • $$a^{-\frac{1}{n}}=\frac{1}{\sqrt[n]{a}}$$
    • $$a^{\frac{m}{n}}=\sqrt[n]{a^m}=(\sqrt[n]{a})^m$$

    Fractional indices result in the nth root of the fraction.

    Distributive Law

    • $$ a(b+c)=ab+ac$$

    When expanding brackets use the distributive law as above.

    Binomial Expressions

    A Binomial Expression is an expression which consists of precisely 2 terms

    For example:

    • $$x+3$$
    • $$a+b$$
    • $$3x+z$$

    When there is a set of 2 binomial expressions multiplied it is called a binomial product.

    For example:

    • $$(x+2)(x-3)$$
    • $$(a+b)(c+d)$$

    When calculating these binomial products, we msut use a technique called "FOIL" which stands for First, Outer, Inner, Last.

    Take for example:

    • $$(x+a)(x+b)$$
      • (First) Multiply the two first terms together.
    • $$(x+a)(x+b)=x^2$$
      • (Outer) Multiply the two outer terms together.
    • $$(x+a)(x+b)=x^2+bx$$
      • (Inner) Multiply the two inner terms together.
    • $$(x+a)(x+b)=x^2+bx+ax$$
      • (Last) Multiply the two last terms together.
    • $$(x+a)(x+b)=x^2+bx+ax+ab$$
      • Then simplify
    • $$(x+a)(x+b)=x^2+x(a+b)+ab$$

    There are some binomial products which have some interesting and useful results.

    Difference of two squares

    • $$(a+b)(a-b)=a^2-b^2$$

    Perfect Squares

    • $$(a+b)^2 = a^2+2ab+b^2$$
    • $$(a-b)^2 = a^2-2ab+b^2$$
